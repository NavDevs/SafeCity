Safe City APK - Build Required

The APK file is not yet available because it needs to be built from the source code.

To Build the APK:

1. Clone the repository:

git clone https://github.com/nas23ise-dot/SafeCity.git

2. Navigate to the project directory:

cd SafeCity

3. Install dependencies:

flutter pub get

4. Build the release APK:

flutter build apk --release

5. The APK will be located at:

build/app/outputs/flutter-apk/app-release.apk

After building, the APK can be downloaded from this location.
