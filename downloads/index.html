Safe City APK - Download Instructions

Note: The APK file is not stored on GitHub due to size limitations (>100MB). You need to build it from the source code.

To Build the APK:

1. Clone the repository:

git clone https://github.com/nas23ise-dot/SafeCity.git

2. Navigate to the project directory:

cd SafeCity

3. Install dependencies:

flutter pub get

4. Build the release APK:

flutter build apk --release

5. The APK will be located at:

build/app/outputs/flutter-apk/app-release.apk

Alternative Installation Methods:

Direct Development Install:

git clone https://github.com/nas23ise-dot/SafeCity.git
cd SafeCity
flutter run
