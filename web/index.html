Prototype Application — Not for Production Use

SafeCity is a college project prototype created for academic evaluation. It is not listed on the Google Play Store or Apple App Store. Features like live emergency dispatch, real-time volunteer routing, and national surveillance integration are simulated for demonstration purposes. The APK is provided solely for academic review and testing.
